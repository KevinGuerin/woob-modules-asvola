# woob-modules-asvola

Modules Woob personnalisés pour TeleCoop et Ethibox.

## telecoop
CapDocument (factures PDF) — API REST Bearer token.

## ethibox
CapDocument (factures PDF) — API REST cookie, PDF généré via reportlab.

## Installation
```bash
cp -r telecoop ethibox ~/.local/share/woob/modules/3.7/woob_modules/
```
